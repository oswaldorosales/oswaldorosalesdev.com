# Oswaldo Rosales | Software Engineer

This is the source code for my personal website [oswaldorosalesdev.com](https://oswaldorosalesdev.com).

## 🛠 Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Self-hosted on Hetzner VPS (managed via Coolify)
- **Language:** TypeScript

## 🚀 Features
- [ ] Responsive design
- [ ] Dark mode support
- [ ] Blog/Project section with Markdown/CMS
- [ ] Contact form with Server Actions

## 🏗 Setup
1. Clone the repo: `git clone ...`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
